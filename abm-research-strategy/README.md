# SKILL 1: ABM RESEARCH & STRATEGY

**Generate comprehensive account research, SWOT analysis, and 14-slide ABM strategy document**

---

## 📋 WHAT THIS SKILL DOES

This skill transforms raw account inputs into a complete ABM strategy package:

**Outputs:**
1. Research Report (PDF, 8-12 pages)
2. ABM Strategy Document (Markdown for Gamma, 14 slides)
3. Backup PPT

**Execution Time:** 45-60 minutes

## 🔧 TROUBLESHOOTING

### Issue: "Missing required input"
**Solution:** Provide all mandatory inputs:
- Account name
- LinkedIn URL
- Website URL
- Contact list Excel file

### Issue: "Research Director QC failed"
**Solution:** The skill found generic statements or unsupported claims. Review the feedback and the skill will refine the research.

### Issue: "MMQQ framework has no real names"
**Solution:** Make sure you uploaded a contact list with actual names and titles.
