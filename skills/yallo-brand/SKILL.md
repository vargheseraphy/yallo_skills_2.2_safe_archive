| Format | Value |

| CMYK | C0 M21 Y69 K17 |
| PMS (approx) | 124 C |

### Colour ratio rule — NON-NEGOTIABLE
- **60%** Black / neutrals
- **30%** Dark surfaces / structure  
- **10%** Gold accent — CTAs, labels, keylines, data highlights, logo

### Gold — what is NEVER allowed
- ✗ Gold as a large background flood
- ✗ Gold as body text
- ✗ Gold on white backgrounds
- ✗ More than one gold focal element per screen

### Supporting context colours (use sparingly — one per layout max)

| Name | HEX | Use |

| Slate Blue | `#3A4A5A` | Intelligence / data context only |

## 3. Typography

### Primary — Plus Jakarta Sans
- Source: Google Fonts
- Import: `https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@300;400;500;600;700;800`
- Use for: ALL headings, body text, UI, wordmark

### Secondary — DM Mono
- Source: Google Fonts
- Import: `https://fonts.googleapis.com/css2?family=DM+Mono:wght@400;500`
- Use for: eyebrows, labels, data, codes, timestamps, `yallo.co` URL in designs

### Type scale

| Role | Font | Size / Weight | Colour |
|---|---|---|---|
| Display / Cover | Plus Jakarta Sans | 48–72px / 800 | #FFFFFF |
| H1 — Section title | Plus Jakarta Sans | 32–42px / 800 | #FFFFFF |
| H2 — Sub heading | Plus Jakarta Sans | 20–24px / 700 | #FFFFFF |
| Body copy | Plus Jakarta Sans | 15–16px / 400 | #F4F4F2 |
| Supporting copy | Plus Jakarta Sans | 13–14px / 300 | #AAAAAA |
| Labels / Eyebrows | DM Mono | 9–11px / 600 uppercase tracked | #D4A843 |
| Code / Mono data | DM Mono | 12px / 400 | #777777 |

### Typography rules
- ✓ Sentence case for headlines always
- ✓ Max 2 lines per heading
- ✓ One clear visual hierarchy per layout
- ✓ Line-height 1.65–1.75 for body
- ✓ Letter-spacing 0.15–0.22em on labels
- ✗ Never all-caps headlines (only 9px labels)
- ✗ Never multiple font weights in one heading
- ✗ Never justified text
- ✗ Never body text below 13px
- ✗ Never exclamation marks in enterprise copy
- ✗ Never more than 2 font families per layout

## 4. Logo System

**Read `references/logo.md` for exact SVG paths.**

### The icon
Four outward-facing D-shaped petal modules meeting at the centre.  
Represents: Strategy, Talent, Technology, People.  
Fill: `#D4A843` · viewBox: `0 0 300 300` · fill-rule: evenodd

### Logo lockups — approved arrangements

1. **Horizontal + GROUP** — standard. Icon left, YALLO right, GROUP below YALLO right-aligned
2. **Horizontal without GROUP** — compact nav/footer use
3. **Vertical + GROUP** — icon top, YALLO + GROUP stacked below, centre-aligned
4. **Compact horizontal** — icon + YALLO + divider + Group (small spaces)
5. **Icon only** — below 20px, favicons, app icons
6. **Hero/large type** — 72px YALLO, icon at proportional scale

### GROUP alignment rule
`display:flex; flex-direction:column; align-items:flex-end` on the text wrapper.
GROUP font: DM Mono, 9–14px, `#D4A843`, letter-spacing: 0.22em, uppercase.

### Approved colour combinations

| Background | Icon | Text |
|---|---|---|
| Black `#000` | Gold `#D4A843` | White `#fff` — **PRIMARY** |
| Dark grey `#1a1a1a` | Gold | White |
| White `#fff` | Gold | Black `#111` — **PRINT** |
| Off-white `#f5f5f3` | Gold | Black |
| Gold `#D4A843` | Black | Black |
| Black | White `#fff` | White — reversed |
| Black | Black `#111` | Black `#111` — 1-colour print |
| White | Grey `#666` | Grey — greyscale |

### Clear space rule
Minimum clear zone = height of the icon on all 4 sides. No text, graphics, or elements within this zone.

### Size guide
| Context | Icon size |
|---|---|
| Hero / banner | 80px |
| Presentation | 48px |
| Standard | 32px |
| Navigation | 20px |
| Email signature | 14px |
| Slide footer | 10px |
| Minimum digital | 16px |
| Minimum print | 5mm |

### Logo — NEVER do
- ✗ Rotate or angle
- ✗ Stretch or distort proportions
- ✗ Recolour (only approved combinations above)
- ✗ Place on non-brand coloured backgrounds
- ✗ Flip or mirror
- ✗ Add drop shadows, glows, or gradients to the logo itself
- ✗ Crowd with other elements (respect clear space)
- ✗ Use as outline/stroke only

- ✗ Carbon fibre (wrong sector)
- ✗ Photographic backgrounds behind text

## 6. Layout Principles

### Gold accent lines
Every YALLO design has a gold gradient top bar (2–3px):
```css
height: 2px;
background: linear-gradient(90deg, #D4A843 0%, #f0c866 45%, transparent 100%);
```

### Presentation slides
- Text always in lower third (never upper)
- Logo + yallo.co in footer left
- Page number in footer right
- Gold top bar + optional left bar
- Grid texture (Pattern B) on all slides
- Icon watermark at 5% opacity (hero/cover) or 4% (content)

### Social posts
- 24px safe zone on all sides — content never touches edge
- Eyebrow label (DM Mono, 9px, gold, tracked uppercase) top-left or top area
- Logo bottom-left + yallo.co bottom-right
- One focal stat or insight per post

### Business cards — print specs
- Size: 55 × 85mm
- Dark: 350gsm silk laminate · CMYK gold: C0 M21 Y69 K17
- Light: 300gsm uncoated
- Canva: https://www.canva.com/design/DAHBjwzLH9w/edit

### Letterhead — print specs
- A4 (210 × 297mm)
- 120gsm paper
- Gold top bar (3–4px)
- Logo watermark in body: 240–260px, 4–5% opacity
- Footer: two lines max — addresses · registration number · yallo.co
- Canva: https://www.canva.com/design/DAGoExbmDsU/edit
