## Creating a Presentation (Pitch Deck)

### Recommended Canva template approach
1. Create new design → Presentation (16:9)
2. Apply YALLO brand kit
3. Use dark background: `#000000`
4. Follow slide structure below

### Slide order for a pitch deck
1. **Cover** — company name + headline + subtitle + confidential label
2. **Problem** — 1 strong pain statement + stat
3. **Solution** — YALLO approach + 72hrs proof
4. **How it works** — 3-step process
5. **Proof** — client logos or anonymised case + metrics
6. **Why YALLO** — 5 proof points (two-column layout)
7. **Team** — Sumeet + key people with titles
8. **Commercial model** — engagement structure
9. **Next steps** — simple 3-step CTA
10. **Cover repeat** — same as slide 1 (closes the deck)

### Text hierarchy in Canva
- Heading 1: Plus Jakarta Sans, ExtraBold (800), 32–42px, White
- Heading 2: Plus Jakarta Sans, Bold (700), 20–24px, White
- Body: Plus Jakarta Sans, Regular (400), 15–16px, #F4F4F2
- Labels: DM Mono, Medium (500), 9–11px, #D4A843, ALL CAPS, tracked

### Colours to set manually in Canva (use hex)
- Primary: `#000000`
- Surface: `#111111`
- Card: `#1F1F1F`
- Gold: `#D4A843`
- White: `#FFFFFF`
- Off-white: `#F4F4F2`
- Muted: `#555555`

### Gold accent line (add to every slide)
- Draw rectangle: full width, 3px tall, position at top y=0
- Fill: gradient left→right: `#D4A843` → `#F0C866` → transparent

### Logo placement on slides
- Footer left: YALLO icon (10px) + "YALLO" (9px, weight 800)
- Footer right: page number in gold DM Mono

## Creating a Social Post

### Canvas size
- Square post: 1080 × 1080px
- LinkedIn recommended: 1200 × 1200px

### Safe zone guide
- Draw guides at 24px from all 4 edges
- ALL content must stay within these guides

### Layer order (bottom to top)
1. Background colour rectangle (`#000000`)
2. Radial glow effect (semi-transparent circle, gold, blur)
3. Grid texture (very subtle, 3–4% opacity)
4. Top gold bar (full width, 2–3px)
5. Content (eyebrow + headline + body)
6. Logo lockup (bottom-left)
7. URL text (bottom-right, DM Mono)

## Creating a LinkedIn Banner

### Canvas size: 1584 × 396px

### Layout zones
- Left 30%: logo + tagline
- Centre: headline or visual
- Right 10%: url / CTA

### Background
- Dark: `#000000` or gradient `#000 → #1f1f1f`
- Gold top bar: 4px, full width
- Optional: Pattern B (fine grid at 4% opacity)

## Using the YALLO Brand Kit in Canva

1. In any design, click the **Brand** tab in the left panel
2. Select **YALLO Group** kit
3. Brand colours will appear: use them instead of the colour picker
4. Brand fonts will be available in the font selector

### If brand kit fonts don't appear
- Search "Plus Jakarta Sans" in the font picker
- Search "DM Mono" in the font picker
- These are free Google Fonts available in Canva

## Common Canva Design Mistakes to Avoid

- ✗ Using the default Canva colour palette — always use YALLO hex values
- ✗ Placing any content touching the edge — always use 24px safe zone
- ✗ Mixing font families beyond Plus Jakarta Sans + DM Mono
- ✗ Using more than 1 gold element as a focal point per slide
- ✗ Using gold as background or body text
- ✗ Making the logo too small to read (minimum 10mm on print)
- ✗ Placing text over a pattern without reducing pattern opacity
- ✗ Centre-justifying body text
- ✗ Using exclamation marks in enterprise copy
