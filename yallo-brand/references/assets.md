# YALLO Asset Templates & Specifications

## Presentation Slides — Dark (standard)

### Cover slide
- Background: `#000000` with radial gold glow at bottom-left
- Gold top bar: 2px gradient `#D4A843 → #f0c866 → transparent`
- Gold left bar: 2px gradient `#D4A843 → transparent` (vertical)
- Icon watermark: 180×180px, `opacity:0.05`, right side, vertically centred
- Grid texture: Pattern B (24px gold grid, 4% opacity)
- Content: lower third (bottom 15% of slide)
- Eyebrow: DM Mono, 8px, gold, tracked uppercase
- Title: Plus Jakarta Sans 800, 28px, white, letter-spacing -.5px
- Subtitle: Plus Jakarta Sans 400, 11px, `#555555`
- Footer: logo+yallo.co left · page number right · border-top `rgba(212,168,67,.1)`

### Content slide (two-column)
- Background: `#080808`
- Left column (42%): eyebrow + large heading + gold divider line + descriptor
- Right column: 5 proof point rows with gold left border `rgba(212,168,67,.06)` background
- Layout: `grid-template-columns: 42% 1fr; align-items: start; gap: 3%`
- Proof point row: `border-left:2px solid #D4A843; border:1px solid rgba(212,168,67,.14); background:rgba(212,168,67,.06); border-radius:2px`
- Number label: DM Mono 7px, weight 700, gold
- Text: Plus Jakarta Sans 9.5px, `#888888`

### Section divider
- Background: `#000000` with centred radial gold glow
- Bottom bar: 2px gold line centred
- Content: perfectly centred, vertical + horizontal
- Section label: DM Mono, 9px, gold, tracked uppercase
- Title: Plus Jakarta Sans 800, 38px, white
- Gold divider: 32px wide, 1.5px, `#D4A843`, margin auto

### Stats slide

## Email Signature — Install Instructions

**Outlook Desktop:**
File → Options → Mail → Signatures → New → paste HTML in edit box

**Outlook Web:**
Settings → Mail → Compose and reply → Signature → paste HTML

**Gmail:**
Settings → All settings → General → Signature → New → click `</>` HTML icon → paste

**Photo hosting:**
Upload headshot to Google Drive → Share → Anyone with link → use direct image URL in src attribute.
